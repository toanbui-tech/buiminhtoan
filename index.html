Getting Started

DevTimeTracker is a standalone Windows executable. No Python installation or dependencies required. Simply download, extract, and run.

System Requirements

  • • Windows 7, 8, 10, 11 or later
  • • 50 MB available disk space
  • • No internet connection required
  • • All data stored locally on your machine

How to use

  1. Download the .exe file using the button above
  2. Double-click to run the application
  3. Create your first project or start tracking immediately
  4. All sessions and data are saved locally
  5. View your time tracking history in the Dashboard tab

💡 First launch may take 5–10 seconds as Windows unpacks the executable. This is normal behavior. Windows Defender may display a warning—click "More info" then "Run anyway".
